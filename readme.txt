==============================================================================================================
使い方
==============================================================================================================
  (1)「フォルダ変更」ボタンを押すとディレクトリ指定のダイアログが開くので、開きたいファイルが入っているフォルダを指定して
     「開く」を押すことで、ディレクトリを指定します。  
      同じディレクトリを使い続けるなら2回目以降は最後に使用したフォルダ情報が保存されるので、この設定は省略できます。  
      
  (2) ディレクトリを指定すると上のプルダウンの中身が自動更新されるので開きたいファイルを選んで「再生」ボタンを押すことで  
  　 音楽ファイルを既定メディアで再生されます。  
    
  (3) プルダウンから現在再生中のもの以外を選んで「再生」ボタンを押すと現在のものに上書きされて再生されます。  
  
  (4) 一番下のテキストボックスで文字を入れて「OK」ボタンを押すと、音楽名を検索できます。アルファベットは大文字・小文字  
  　 の区別はされません。日本語での検索もできます。  
    
=============================================================================================================
注意
=============================================================================================================
　・音楽ファイル以外もプルダウンに表示されますが、音楽ファイル以外のファイルは実行できません。
 　 拡張子によっては実行できるものがあるかもしれませんが、正常な動作は保証できません。
  ・MusicSelector.exeと同じ階層にPath.txtがないと動作しません。
